## 项目结构
```text
AIDevHub/
├─ src/
│  └─ aidevhub/
│     ├─ __init__.py        # 包入口
│     └─ cli.py             # 命令行入口
├─ docs/
│  ├─ theory/               # AI 理论与提示工程相关笔记
│  ├─ papers/               # 参考论文与资料
│  └─ screenshots/          # 文档配图
├─ examples/
│  └─ eg_prompt_basic.py    # Prompt 基础示例
├─ tests/
│  └─ __init__.py           # 测试目录
├─ .env.example             # 环境变量示例
├─ pyproject.toml           # 项目配置与依赖声明
├─ uv.lock                  # 依赖锁定文件
└─ README.md                # 项目说明
```
